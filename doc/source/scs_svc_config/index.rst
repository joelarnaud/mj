Module scs_svc_config
=====================

Description : Ce module a pour but d'activer le service config d'AWS qui  sert à enregistrer les différents états de configuration des objets
              dans nos environnements AWS, que ces changements de configuration soient effectués dans la console AWS ou via les scripts TF. Il permet également de mettre en place des alertes relatives à des configurations dangeureuses.
