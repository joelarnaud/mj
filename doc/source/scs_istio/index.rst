Module scs_istio
================

Description : Ce module permet d'installer `istio`_ dans un cluster kubernetes. Il n'est cependant pas automatiquement activé sur les namespaces.
Istio permet de la mise en place d'un service mesh au sein d'un cluster kubernetes.
